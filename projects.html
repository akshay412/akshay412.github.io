Continual Learning for Autonomous Vehicles

In this project, I explored and implemented various continual learning algorithms aimed at improving steering control in autonomous vehicles. By continuously learning from new experiences while retaining past knowledge, this framework aims to enable autonomous vehicles to navigate diverse environments with improved accuracy and robustness, thus advancing the capabilities of next-generation transportation systems.

Key Features

  • Implemented multiple continual learning algorithms, including:
    • Elastic Weight Consolidation (EWC)
    • Experience Replay Buffer
    • A novel Temporal Consistency Regularization (TCR) technique with a balanced buffer
  • Developed a VisualBackProp implementation for feature importance visualization

My main innovation was the Temporal Consistency Regularization technique. TCR utilizes a balanced buffer along with leveraging the temporal aspects of the problem, significantly enhancing prediction accuracy and driving smoothness. This method regularization method takes advantage of the temporal nature of driving while continually learning from different distributions of data

Multimodal Sentiment Analysis

This project delves into different methodologies for Multimodal Sentiment Analysis, with a focus on fusion techniques that integrate information from various modalities to improve sentiment analysis performance, as well as the architectures employing them.

Key Features

  • Utilizes CMU-MOSI and CMU-MOSEI datasets for sentiment analysis
  • Implements various fusion techniques and model architectures:
    • Early Fusion (GRU and Transformer)
    • LateFusion (GRU and Transformer)
    • Tensor Fusion
    • Low Rank Tensor Fusion
    • Memory Fusion Model (MFM)
    • Multimodal Cyclic Translation Network (MCTN)
    • Multimodal Transformer (MulT)
  • Comparative study of all the models

Each approach presents unique advantages and challenges, offering a range of tools to address the intricacies of sentiment analysis in multimodal data. This project demonstrates the effectiveness of combining multiple modalities for sentiment analysis, with transformer-based models generally outperforming other architectures.

Financial Fraud Detection Framework

This project implements and optimizes graph neural networks for credit card fraud detection

Key Features

  • Implemented a Temporal Graph Attention (TGAT) mechanism for semi-supervised credit card fraud detection
  • Developed an architecture combining:
    • Categorical and numerical attributes
    • Multi-head attention for neighbor importance
    • Attribute-driven gated residuals for enhanced interpretability and performance
  • Optimized data processing and model training:
    • Reduced data preprocessing time by 90% using numba and cython
    • Reduced model training time by 22% using cuda-numba

This project demonstrates the effectiveness of graph neural networks in detecting financial fraud , with a focus on optimizing performance and interpretability through innovative architectural designs and efficient implementation techniques.
